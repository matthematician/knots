Closed: A curve is closed if the function \({\bf r}\colon \mathbb{R}\to\mathbb{R}^3\) is periodic, that is, if there exists a positive real number \(T\) for which,

\begin{equation*} \text{for all }t\in \mathbb{R}, \qquad \text{we have } {\bf r}\bigl(t+T\bigr) = {\bf r}(t). \end{equation*}

The smallest such \(T\) is called the period of the function \({\bf r}\text{,}\) and we think of a closed curve as one whose “starting point” and “ending point” are the same, \({\bf r}(0) = {\bf r}(T).\)
