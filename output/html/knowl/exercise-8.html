Exercise 2.

Consider the \(2\times 2\) matrix

\begin{equation*} X_t = \left[\begin{array}{cc} 1-t\amp t\\ 1\amp 0\end{array}\right]. \end{equation*}

  1. Form the matrix \(X_0\) by substituting \(0\) for \(t\text{.}\) Then use matrix multiplication to show that \(X_0\cdot X_0 = \left[\begin{array}{rr}1\amp 0\\0\amp 1\end{array}\right]\text{,}\) which is the \(2\times 2\) identity matrix.

  2. Compute the matrix product \(X_t\cdot X_t\text{.}\) Is this an identity matrix? What's different?

  3. If we place the matrix \(X_t\) into the upper and lower blocks of a \(3\times 3\) matrix, we obtain

    \begin{equation*} M = \left[\begin{array}{ccc} 1-t\amp t\amp \color{gray}{0}\\ 1\amp 0 \amp \color{gray}{0}\\ \color{gray}{0}\amp \color{gray}{0}\amp \color{gray}{1} \end{array}\right]\text{ and } N = \left[\begin{array}{ccc} \color{gray}{1}\amp \color{gray}{0}\amp \color{gray}{0}\\ \color{gray}{0}\amp 1-t\amp t\\ \color{gray}{0}\amp 1 \amp 0\end{array}\right]. \end{equation*}

    Use Sage to compute both the matrix products \(M\cdot N \cdot M\) and \(N\cdot M \cdot N\text{.}\) What do you notice?

  4. Show how we can use the matrices \(M\) and \(N\) to represent the braid

    \begin{equation*} b = s_1^2s_2s_1^3s_2 \end{equation*}

    and then use that matrix to determine the induced permutation \(F(b)\text{.}\)
