We saw in Section 3.1 that the same knot can have (infinitely!) many different diagrams. Yet we want to be able to study the properties of knots, which reside in three dimensions, using diagrams which can obscure knots' three-dimensional reality. Definition 3.1.7 showed exactly how isotopy of a knot, which leaves the knot unchanged, can nevertheless change its diagrams in very specific ways.
