Definition 1.1.1.

An \(n\)-ary operation on a set \(S\) is a function which assigns to every “input” collection of \(n\) elements of \(S\text{,}\) an “output” element of \(S\text{.}\)

When \(n=1\) we call an operation unary.

When \(n=2\) we call an operation binary.

When \(n=3\) we call an operation ternary.
