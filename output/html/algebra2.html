Section 1.2 Permutations and Presentations

Objectives

  1. Simplify, solve, and describe algebraic structure in subgroups of permutation groups, using “braid-a-grams”, cycle notation, and/or the Coxeter presentation.

One of the richest algebraic structures with a finite set of elements is that which describes permutations (rearrangements) of those elements. We can begin by thinking of this structure in terms of anagrams of English words.

Definition 1.2.1.

Let \(X = \{1,2,\ldots,n\}\) be an enumerated finite set. A permutation of n symbols is a bijective function \(\sigma \colon X\to X\text{.}\)

In other words, a permutation of \(n\) symbols is a way of rearranging all the numbers \(1\) through \(n\) (called the “positions” of the permutation) so that no two positions end up in the same spot:

\begin{equation*} \text{If } i\neq j \text{ then } \sigma(i) \neq \sigma(j). \end{equation*}
Example 1.2.2.

An example of a permutation of five symbols might be the function \(\sigma \colon \{1,2,3,4,5\} \to \{1,2,3,4,5\}\) for which

\begin{align*} \sigma(1) \amp= 2\\ \sigma(2) \amp= 1\\ \sigma(3) \amp= 5\\ \sigma(4) \amp= 3\\ \sigma(5) \amp= 4. \end{align*}

Note that no two “starting” positions have the same “ending” position.

If instead of the numbers \(1\) through \(5\) we had used the letters of the word \(ALERT\) as the elements of our set, we would obtain

\begin{align*} \sigma(A) \amp= L\\ \sigma(L) \amp= A\\ \sigma(E) \amp= T\\ \sigma(R) \amp= E\\ \sigma(T) \amp= R. \end{align*}

In this way, anagrams of English words can be understood as permutations on the set of its letters. (We will typically use words that do not have any repeated letters -- otherwise, we might be unable to define the permutation uniquely!)

You may remember from finite mathematics that there are \(n! = n(n-1)(n-2)\cdots(2)(1)\) different permutations for a set of \(n\) symbols. Taken together with the composition of functions operation -- that is, perform one permutation and then another permutation, using the result of the former as the input for the latter -- we have an algebraic structure.

Definition 1.2.3.

Let \(S_n\) denote the set of all permutations on \(n\) symbols. Together with the composition of functions operation \(\circ : (\sigma,\tau) \mapsto \sigma\circ\tau\text{,}\) we call \((S_n,\circ)\) the permutation group on n symbols.

You might notice that Definition 1.2.3 is more than a definition -- it is a theorem in disguise! This is because of the word group: the magma \((S_n,\circ)\) is associative, it has the identity property via the “trivial permutation”

\begin{gather} \epsilon(1) = 1, \quad \epsilon(2)=2, \quad \ldots \quad, \epsilon(n)=n\text{,}\label{eq_identitypermutation}\tag{1.2.1} \end{gather}

and it has the invertibility property: anything a permutation does, can be undone.

Example 1.2.4. Composition, identity, inverse permutations.

The permutation \(\sigma \in S_4\) defined by

\begin{equation*} \sigma(1)=3,\quad \sigma(2)=4,\quad \sigma(3)=2, \quad \sigma(4)=1 \end{equation*}

performs, for example, the anagram \(TEAM \mapsto MATE\text{.}\)

We can perform the composition \(\sigma\circ \sigma\) by applying the permutation \(\sigma\) to the result of having applied \(\sigma\text{:}\)

If we write \(\tau = \sigma\circ\sigma\) for this permutation, then we have

\begin{equation*} \tau(1)=2,\quad \tau(2)=1,\quad \tau(3)=4,\quad \tau(4)=3. \end{equation*}

The diagrams you sketched in Activity 1.2.2 are called “braid-a-grams”, and give us a convenient way to analyze and dissect permutations.

Definition 1.2.8.

Let \(\sigma \in S_n\) be a permutation on \(n\) symbols. A braid-a-gram for \(\sigma\) is a diagram in which each initial position \(i\) is joined by a curve to its final position \(\sigma(i)\text{,}\) and in which each crossing of two curves occurs at a different height in the diagram.

Definition 1.2.10.

A permutation \(\sigma \in S_n\) is called a \(k\)-cycle if there exists a sequence of \(k\) positions \(x_1,x_2,\ldots,x_k\) such that

\begin{equation*} \sigma(x_1) = x_2, \quad \sigma(x_2) = x_3, \quad \ldots , \sigma(x_{k-1}) = x_k, \quad \sigma(x_k) = x_1. \end{equation*}

We denote the above \(k\)-cycle using cycle notation as

\begin{equation*} \sigma = \bigl( x_1\, x_2\, \cdots\, x_k \bigr). \end{equation*}

When \(k=2\) we call the cycle a transposition.

For example, the anagram \(ALERT \mapsto LATER\) from Example 1.2.2 can be expressed in cycle notation as \(\sigma = (12)(345)\text{.}\)

The main result we're seeing here is the following theorem.

Theorem 1.2.11.

Every permutation of \(n\) symbols can be written as a composition of adjacent transpositions.

Since the adjacent transpositions play such an important role in permutations, we'll use the shorthand

\begin{equation*} \sigma_i = \bigl(i\;i+1\bigr) \end{equation*}

to denote the transposition that swaps the \(i\)-th position with its neighbor to the right.

Theorem 1.2.12. Coxeter Presentation.

Let \(S_n\) be the permutation group on \(n\) symbols, and denote by \(\sigma_i = ( i, i+1 )\) the \(i\)-th adjacent transposition for \(1 \leq i \leq n-1\text{.}\)

Then every permutation \(\sigma\in S_n\) can be expressed as a product
\begin{equation*} \sigma = \sigma_{i_1}\circ \sigma_{i_2} \circ \cdots \circ \sigma_{i_k}. \end{equation*}
Furthermore, the transpositions \(\sigma_i\) satisfy the following relations.
\begin{align} \sigma_i^2 \amp= \epsilon \amp\amp \text{for all }1 \leq i \leq n-1.\label{eq_cox1}\tag{1.2.2}\\ \sigma_i\sigma_j \amp= \sigma_j\sigma_i \amp\amp \text{for all }|i-j| \geq 2.\label{eq_cox2}\tag{1.2.3}\\ \sigma_i\sigma_{i+1}\sigma_i \amp= \sigma_{i+1}\sigma_i\sigma_{i+1} \amp\amp \text{for all }1 \leq i \leq n-2.\label{eq_cox3}\tag{1.2.4} \end{align}
